### 3.1 工具函数

#### generateRoomCode()

**功能**: 生成6位房间号

**代码位置**: server.js:93-100

**返回值**: string - 6位大写字母+数字组合

**实现逻辑**:
```javascript
function generateRoomCode() {
    const chars = 'ABCDEFGHJKLMNPQRSTUVWXYZ23456789';
    let code = '';
    for (let i = 0; i < 6; i++) {
        code += chars.charAt(Math.floor(Math.random() * chars.length));
    }
    return code;
}
```

**注意**: 排除了容易混淆的字符 I, O, 0, 1

### 4.2 玩家管理

#### addPlayer(socket, username, avatar, voice)

**功能**: 添加玩家到房间

**代码位置**: server.js:307-458

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|

| username | string | 玩家名称 |
| avatar | string | 头像emoji |
| voice | string | 语音类型 |

**返回值**: Player | null

**处理流程**:
```
1. 检查是否是重连（相同用户名的离线玩家）

   │   ├── 更新socket和offline状态
   │   ├── 取消自动解散计时器

   │   └── 如果游戏进行中，发送游戏状态
   └── 不是重连 → 继续检查
   
2. 检查房间是否已满
   └── 已满 → 返回null

3. 创建新玩家对象
   ├── 分配座位索引
   ├── 设置风位
   ├── 设置房主标记
   └── 初始化游戏数据

4. 添加到玩家列表
5. 更新playerSockets映射
6. 广播房间更新
```

#### addAIPlayer()

**功能**: 添加AI玩家

**代码位置**: server.js:461-503

**返回值**: Player | null

**实现逻辑**:
```javascript
addAIPlayer() {
    if (this.players.length >= 4) return null;
    
    const seatIndex = this.players.length;
    const aiNames = ['AI小明', 'AI小红', 'AI小刚', 'AI小丽'];
    const aiAvatars = ['🤖', '🎮', '💻', '🎯'];
    
    // 动态分配语音，避开已有玩家的语音
    const allVoices = ['female01', 'female02', 'male', 'male02'];
    const usedVoices = this.players.map(p => p.voice);
    const availableVoices = allVoices.filter(v => !usedVoices.includes(v));
    const aiVoice = availableVoices.length > 0 
        ? availableVoices[0] 
        : allVoices[seatIndex % 4];
    
    const aiPlayer = {
        id: 'ai_' + Date.now() + '_' + seatIndex,
        username: aiNames[seatIndex] || 'AI 玩家',
        avatar: aiAvatars[seatIndex] || '🤖',
        voice: aiVoice,
        socket: null,
        ready: true,  // AI默认准备
        // ... 其他属性
    };
    
    this.players.push(aiPlayer);
    this.broadcastRoomUpdate();
    return aiPlayer;
}
```

   ├── 从列表中移除
   ├── 重新分配座位
   ├── 转移房主（如果需要）
   └── 如果无真人玩家，解散房间
```

// 所有真人玩家离线后30秒自动解散

setPlayerReady(socketId, ready) {
    const player = this.players.find(p => p.id === socketId);
    if (player) {
        player.ready = ready;
        player.aiTakeover = false;  // 玩家主动准备，取消AI接管
        
        // 如果在倒计时中
        if (this.nextRoundTimer) {
            this.broadcastReadyStatus();
            
            // 检查是否全员准备
            const allReady = this.players.every(p => p.ready);
            if (allReady) {
                clearInterval(this.nextRoundTimer);
                this.nextRoundTimer = null;
                setTimeout(() => {
                    if (!this.gameRunning) {
                        this.startGame();
                    }
                }, 500);
            }
        } else {
            this.broadcastRoomUpdate();
            this.checkCanStart();
        }
    }
}
```

checkCanStart() {
    const realPlayers = this.players.filter(p => !p.isBot);
    const allReady = realPlayers.every(p => p.ready);
    
    if (allReady && realPlayers.length >= 1 && !this.gameRunning) {
        this.fillWithAI();  // 填充AI到4人

### 5.2 加入房间

**事件**: `join_room`

**代码位置**: server.js:3786-3859

**处理流程**:
```
1. 获取房间号并转大写
2. 检查房间是否存在
   ├── TEST房间不存在则自动创建
   └── 其他房间不存在则返回错误
   
3. 检查是否是断线重连
   └── 是 → 恢复玩家状态

4. 检查游戏是否进行中
   ├── 房间满了 → 以观战者身份加入
   └── 房间没满 → 给玩家选择

5. 检查真人玩家数量
   └── 已满4人 → 返回错误

6. 如果有AI，踢掉一个AI腾位置
7. 添加玩家
8. 发送成功事件
```
