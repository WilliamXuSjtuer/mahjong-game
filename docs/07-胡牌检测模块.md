# 胡牌检测模块

本文档详细描述胡牌检测和牌型验证相关的功能实现。

## 一、模块概述

胡牌检测模块负责验证玩家是否可以胡牌，以及检测各种特殊牌型（七对子、碰碰胡、清一色等）。

## 二、胡牌基本条件

### 2.1 胡牌牌型

麻将胡牌的基本结构是 **4组面子 + 1对将**：

```
面子类型:
- 顺子: 三张连续的同花色牌（如123万）
- 刻子: 三张相同的牌（如三万三万三万）
- 杠子: 四张相同的牌（算作一组面子）

将: 两张相同的牌
```

### 2.2 特殊牌型

- **七对子**: 7个对子，共14张牌
- **碰碰胡**: 4组刻子/杠子 + 1对将

### 3.1 canHu(hand, melds)

**功能**: 检查是否可以胡牌

**代码位置**: server.js:2650-2662

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| hand | Tile[] | 手牌数组 |
| melds | Meld[] | 副露数组 |

**返回值**: boolean

```javascript
canHu(hand, melds) {
    // 检查是否有14张牌（或11/8/5张+副露）
    const totalTiles = hand.length + melds.length * 3;
    if (totalTiles !== 14) return false;
    
    // 检查七对子
    if (hand.length === 14 && this.isQiDui(hand)) {
        return true;
    }
    
    // 简化版胡牌检测：3N+2结构
    return this.checkWinningHand([...hand]);
}
```

### 3.2 checkWinningHand(tiles)

**功能**: 验证胡牌牌型

**代码位置**: server.js:2700-2722

**算法**: 递归尝试找出将牌，然后验证剩余牌能否组成面子

```javascript
checkWinningHand(tiles) {
    if (tiles.length === 0) return true;
    if (tiles.length === 2) {
        // 最后剩两张，必须是对子（将牌）
        return tiles[0].type === tiles[1].type && 
               tiles[0].value === tiles[1].value;
    }
    if (tiles.length < 3) return false;
    
    const sorted = sortTiles(tiles);
    
    // 尝试作为将（对子）
    for (let i = 0; i < sorted.length - 1; i++) {
        if (sorted[i].type === sorted[i+1].type && 
            sorted[i].value === sorted[i+1].value) {
            const remaining = [...sorted];
            remaining.splice(i, 2);  // 移除对子
            if (this.canFormMelds(remaining)) {
                return true;
            }
        }
    }
    
    return false;
}
```

### 3.3 canFormMelds(tiles)

**功能**: 验证剩余牌能否组成面子

**代码位置**: server.js:2724-2758

**算法**: 递归尝试组成刻子或顺子

```javascript
canFormMelds(tiles) {
    if (tiles.length === 0) return true;
    if (tiles.length % 3 !== 0) return false;
    
    const sorted = sortTiles(tiles);
    
    // 尝试刻子（三张相同）
    if (sorted.length >= 3 &&
        sorted[0].type === sorted[1].type && 
        sorted[1].type === sorted[2].type &&
        sorted[0].value === sorted[1].value && 
        sorted[1].value === sorted[2].value) {
        const remaining = sorted.slice(3);
        if (this.canFormMelds(remaining)) return true;
    }
    
    // 尝试顺子（三张连续）
    if (sorted.length >= 3) {
        const first = sorted[0];
        
        // 找第二张（value + 1）
        const secondIdx = sorted.findIndex(t => 
            t.type === first.type && t.value === first.value + 1
        );
        
        // 找第三张（value + 2）
        const thirdIdx = sorted.findIndex(t => 
            t.type === first.type && t.value === first.value + 2
        );
        
        if (secondIdx !== -1 && thirdIdx !== -1) {
            const remaining = [...sorted];
            // 从大索引开始移除，避免索引错位
            const indices = [0, secondIdx, thirdIdx].sort((a, b) => b - a);
            indices.forEach(idx => remaining.splice(idx, 1));
            if (this.canFormMelds(remaining)) return true;
        }
    }
    
    return false;
}
```

## 四、特殊牌型检测

### 4.1 七对子 (isQiDui)

**功能**: 检测七对子

**代码位置**: server.js:2573-2582

**条件**: 7个对子，每种牌恰好2张

```javascript
isQiDui(hand) {
    const counts = {};
    hand.forEach(tile => {
        const key = `${tile.type}${tile.value}`;
        counts[key] = (counts[key] || 0) + 1;
    });
    
    const values = Object.values(counts);
    return values.length === 7 && values.every(v => v === 2);
}
```

**示例**:
```
手牌: [一万x2, 三万x2, 五万x2, 七万x2, 二条x2, 四条x2, 东风x2]
统计: {一万:2, 三万:2, 五万:2, 七万:2, 二条:2, 四条:2, 东风:2}
结果: 7种牌，每种2张 → 七对子 ✓
```

### 4.2 碰碰胡 (isPengPengHu / checkPengPengHu)

**功能**: 检测碰碰胡

**代码位置**: server.js:2584-2603, 2846-2856

**条件**: 4组刻子/杠子 + 1对将

```javascript
isPengPengHu(hand) {
    const counts = {};
    hand.forEach(tile => {
        const key = `${tile.type}${tile.value}`;
        counts[key] = (counts[key] || 0) + 1;
    });
    
    let pairCount = 0;
    let tripleCount = 0;
    
    for (const key in counts) {
        const count = counts[key];
        if (count === 2) pairCount++;
        else if (count === 3) tripleCount++;
        else if (count === 4) tripleCount++;  // 杠子算刻子
        else if (count === 1) return false;   // 孤张不行
    }
    
    return pairCount === 1 && tripleCount >= 4;
}

checkPengPengHu(hand, melds) {
    // 检查副露是否都是刻子或杠
    for (const meld of melds) {
        if (meld.type !== 'peng' && meld.type !== 'gang') {
            return false;
        }
    }
    
    // 检查手牌是否能组成全刻子+一对将
    return this.canFormAllPeng(hand);
}
```

### 4.3 canFormAllPeng(tiles, depth, hasJiang)

**功能**: 检查手牌是否能组成全刻子

**代码位置**: server.js:2859-2892

```javascript
canFormAllPeng(tiles, depth = 0, hasJiang = false) {
    if (tiles.length === 0) {
        return hasJiang;
    }
    if (tiles.length === 2) {
        if (hasJiang) return false;
        return tiles[0].type === tiles[1].type && 
               tiles[0].value === tiles[1].value;
    }
    if (tiles.length < 3) return false;
    
    const sorted = sortTiles(tiles);
    
    // 尝试将第一组作为刻子
    if (sorted.length >= 3 &&
        sorted[0].type === sorted[1].type && 
        sorted[1].type === sorted[2].type &&
        sorted[0].value === sorted[1].value && 
        sorted[1].value === sorted[2].value) {
        const remaining = sorted.slice(3);
        if (this.canFormAllPeng(remaining, depth + 1, hasJiang)) {
            return true;
        }
    }
    
    // 尝试将前两张作为将牌（如果没有将牌）
    if (!hasJiang && sorted.length >= 2 &&
        sorted[0].type === sorted[1].type &&
        sorted[0].value === sorted[1].value) {
        const remaining = sorted.slice(2);
        if (this.canFormAllPeng(remaining, depth + 1, true)) {
            return true;
        }
    }
    
    return false;
}
```

### 4.4 清一色 (isQingYiSe)

**功能**: 检测清一色

**代码位置**: server.js:2605-2625

**条件**: 只有一种花色，无字牌

```javascript
isQingYiSe(hand, melds = []) {
    const suits = new Set();
    let hasHonor = false;
    
    const allTiles = [...hand];
    for (const meld of melds) {
        if (meld.tiles) {
            allTiles.push(...meld.tiles);
        }
    }
    
    allTiles.forEach(tile => {
        if (['wan', 'tiao', 'tong'].includes(tile.type)) {
            suits.add(tile.type);
        } else if (['feng', 'zhong', 'fa', 'bai', 'honor', 'wind'].includes(tile.type)) {
            hasHonor = true;
        }
    });
    
    return suits.size === 1 && !hasHonor;
}
```

**示例**:
```
清一色（万子）: [一万, 二万, 三万, 四万, 五万, 六万, 七万, 八万, 九万, 一万, 一万, 二万, 二万, 二万]
清一色（条子）: [一条, 二条, 三条, ...] 全部条子
清一色（筒子）: [一筒, 二筒, 三筒, ...] 全部筒子
```

### 5.1 getTingTiles(hand, melds)

**功能**: 获取听牌列表

**代码位置**: server.js:2665-2698

**返回值**: TingTile[] - 可以胡的牌列表

```javascript
getTingTiles(hand, melds = []) {
    const tingTiles = [];
    
    // 数字牌：万、条、筒
    for (const type of ['wan', 'tiao', 'tong']) {
        for (let value = 1; value <= 9; value++) {
            const testTile = { type, value };
            const testHand = [...hand, testTile];
            if (this.canHu(testHand, melds)) {
                tingTiles.push({ 
                    type, 
                    value, 
                    tileName: getTileName(testTile) 
                });
            }
        }
    }
    
    // 字牌：风向
    for (const windValue of WINDS) {
        const testTile = { type: 'wind', value: windValue };
        const testHand = [...hand, testTile];
        if (this.canHu(testHand, melds)) {
            tingTiles.push({ 
                type: 'wind', 
                value: windValue, 
                tileName: getTileName(testTile) 
            });
        }
    }
    
    // 箭牌：中发白
    for (const honorValue of ['zhong', 'fa', 'bai']) {
        const testTile = { type: 'honor', value: honorValue };
        const testHand = [...hand, testTile];
        if (this.canHu(testHand, melds)) {
            tingTiles.push({ 
                type: 'honor', 
                value: honorValue, 
                tileName: getTileName(testTile) 
            });
        }
    }
    
    return tingTiles;
}
```

**示例**:
```
手牌: [一万, 一万, 二万, 三万, 四万, 五万, 六万, 七万, 八万, 九万, 东风, 东风, 东风]
```

## 八、算法复杂度分析

### 8.1 canHu 复杂度

- **最坏情况**: O(n!) - 递归尝试所有组合
- **实际平均**: O(n) - 牌型通常很快确定

### 8.2 优化建议

1. **预计算**: 预先统计每种牌的数量
2. **剪枝**: 发现不可能的组合立即返回
3. **缓存**: 对相同牌型缓存结果

### 8.3 当前实现特点

- 递归回溯算法
- 优先检测七对子（快速判断）
- 支持任意牌型组合
