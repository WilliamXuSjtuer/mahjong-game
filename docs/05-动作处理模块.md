# 动作处理模块

本文档详细描述麻将游戏中的动作处理逻辑，包括碰、杠、吃、胡等操作。

## 一、模块概述

动作处理模块负责处理玩家在游戏中可以执行的各种动作，包括：
- **碰**: 三张相同的牌组成刻子
- **杠**: 四张相同的牌组成杠子（明杠、暗杠、加杠）
- **吃**: 三张连续的牌组成顺子
- **胡**: 胡牌获胜

## 二、动作优先级

```
胡 > 杠 > 碰 > 吃 > 过

优先级值:
{
    hu: 4,
    gang: 3,
    peng: 2,
    chi: 1,
    pass: 0
}
```

## 三、动作检测流程

### 3.1 checkActionsAfterDiscard(tile, discardPlayerIndex)

**功能**: 检查出牌后其他玩家可以执行的动作

**代码位置**: server.js:1440-1519

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| tile | Tile | 打出的牌 |
| discardPlayerIndex | number | 出牌玩家索引 |

**处理流程**:

```javascript
checkActionsAfterDiscard(tile, discardPlayerIndex) {
    this.gameState.pendingActions = [];
    
    for (let i = 0; i < 4; i++) {
        if (i === discardPlayerIndex) continue;  // 跳过出牌者
        
        const player = this.players[i];
        if (!player) continue;
        
        const actions = [];
        
        // 1. 检查胡牌（只有敲牌后才能胡）
        const testHand = [...player.hand, tile];
        if (player.isQiao && this.canHu(testHand, player.melds)) {
            actions.push('hu');
        }
        
        // 2. 检查杠（有3张相同的牌）
        const sameCount = player.hand.filter(t => 
            t.type === tile.type && t.value === tile.value
        ).length;
        if (sameCount === 3) {
            actions.push('gang');
        }
        
        // 3. 检查碰（有2张相同的牌，且未听牌）
        if (sameCount >= 2 && !player.isTing) {
            actions.push('peng');
        }
        
        // 4. 检查吃牌（只能吃上家的牌，且未听牌）
        if (!player.isTing) {
            const isNextPlayer = (discardPlayerIndex + 1) % 4 === i;
            if (isNextPlayer && this.canChi(tile, player.hand)) {
                const chiOptions = this.getChiOptions(tile, player.hand, discardPlayerIndex);
                if (chiOptions.length > 0) {
                    actions.push('chi');
                }
            }
        }
        
        if (actions.length > 0) {
            this.gameState.pendingActions.push({
                playerIndex: i,
                playerId: player.id,
                actions: actions,
                tile: tile,
                chiOptions: actions.includes('chi') ? chiOptions : undefined
            });
        }
    }
    
    if (this.gameState.pendingActions.length > 0) {
        // 有玩家可以执行动作
        this.gameState.turnPhase = 'action';
        this.notifyPendingActions();
        
        // 设置超时
        this.gameState.actionTimeout = setTimeout(() => {
            this.resolveActions();
        }, GAME_TIMEOUT_CONFIG.ACTION_TIMEOUT_MS);  // 60秒
    } else {
        // 没有动作，轮到下家
        this.nextTurn();
    }
}
```

### 3.2 notifyPendingActions()

**功能**: 通知等待动作的玩家

**代码位置**: server.js:1584-1625

```javascript
notifyPendingActions() {
    let hasHumanPending = false;
    
    this.gameState.pendingActions.forEach(action => {
        const player = this.players[action.playerIndex];
        
        if (player.isBot) {
            // AI决策（延迟执行）
            setTimeout(() => {
                if (this.gameRunning && !action.resolved) {
                    this.aiDecideAction(player, action);
                }
            }, 500 + Math.random() * 1000);
        } else if (player.offline || !player.socket || player.aiTakeover) {
            // 离线玩家自动过
            action.resolved = true;
            action.action = 'pass';
        } else {
            // 真人玩家
            hasHumanPending = true;
            const emitData = {
                actions: action.actions,
                tile: action.tile
            };
            if (action.chiOptions) {
                emitData.chiOptions = action.chiOptions;
            }
            player.socket.emit('action_available', emitData);
        }
    });
    
    this.broadcastGameState();
    
    // 如果没有真人等待，立即解析
    if (!hasHumanPending && this.gameState.pendingActions.every(a => a.resolved)) {
        clearTimeout(this.gameState.actionTimeout);
        setTimeout(() => this.resolveActions(), 100);
    }
}
```

## 四、动作执行

### 4.1 playerAction(socketId, actionType, extraData)

**功能**: 玩家执行动作

**代码位置**: server.js:1628-1718

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| socketId | string | 玩家Socket ID |
| actionType | string | 动作类型: 'hu', 'gang', 'peng', 'chi', 'pass', 'hu_zimo', 'an_gang', 'jia_gang' |
| extraData | object | 额外数据（如吃牌选项索引） |

**处理逻辑**:

```javascript
playerAction(socketId, actionType, extraData = {}) {
    const player = this.players.find(p => p.id === socketId);
    if (!player) return { error: '玩家不存在' };
    
    // 处理自摸胡牌
    if (actionType === 'hu_zimo') {
        if (this.gameState.pendingZimo && this.gameState.pendingZimo.playerId === socketId) {
            if (this.gameState.discardTimeout) {
                clearTimeout(this.gameState.discardTimeout);
                this.gameState.discardTimeout = null;
            }
            this.endRound('hu', player.seatIndex, -1, true, false);
            this.gameState.pendingZimo = null;
            return { success: true };
        }
        return { error: '不能自摸胡牌' };
    }
    
    // 处理暗杠
    if (actionType === 'an_gang') {
        if (this.gameState.pendingAnGang && this.gameState.pendingAnGang.playerId === socketId) {
            if (this.gameState.discardTimeout) {
                clearTimeout(this.gameState.discardTimeout);
            }
            this.executeAnGang(player, extraData.anGangOptions || this.gameState.pendingAnGang.anGangOptions);
            this.gameState.pendingAnGang = null;
            return { success: true };
        }
        return { error: '不能暗杠' };
    }
    
    // 处理加杠
    if (actionType === 'jia_gang') {
        // ... 类似暗杠处理
    }
    
    // 处理普通动作（碰/杠/吃/胡/pass）
    const pendingAction = this.gameState.pendingActions.find(a => a.playerId === socketId);
    if (!pendingAction) {
        return { error: '没有可执行的动作' };
    }
    
    if (actionType === 'pass') {
        pendingAction.resolved = true;
        pendingAction.action = 'pass';
    } else if (pendingAction.actions.includes(actionType)) {
        pendingAction.resolved = true;
        pendingAction.action = actionType;
        
        // 保存吃牌选项索引
        if (actionType === 'chi' && extraData.selectedChiIndex !== undefined) {
            pendingAction.selectedChiIndex = extraData.selectedChiIndex;
        }
    } else {
        return { error: '无效的动作' };
    }
    
    // 检查是否所有动作都已处理
    if (this.gameState.pendingActions.every(a => a.resolved)) {
        clearTimeout(this.gameState.actionTimeout);
        this.resolveActions();
    }
    
    return { success: true };
}
```

### 4.2 resolveActions()

**功能**: 解析所有动作，执行优先级最高的

**代码位置**: server.js:1721-1752

```javascript
resolveActions() {
    // 通知所有玩家隐藏动作按钮
    this.broadcast('action_timeout', {});
    
    // 将所有未处理的动作自动标记为 pass
    for (const action of this.gameState.pendingActions) {
        if (!action.resolved) {
            action.resolved = true;
            action.action = 'pass';
        }
    }
    
    // 优先级：胡 > 杠 > 碰 > 吃 > pass
    const priority = { hu: 4, gang: 3, peng: 2, chi: 1, pass: 0 };
    
    let bestAction = null;
    for (const action of this.gameState.pendingActions) {
        const actionPriority = priority[action.action] || 0;
        if (!bestAction || actionPriority > priority[bestAction.action]) {
            bestAction = action;
        }
    }
    
    if (bestAction && bestAction.action !== 'pass') {
        this.executeAction(bestAction);
    } else {
        this.nextTurn();
    }
    
    this.gameState.pendingActions = [];
}
```

canChi(tile, hand) {

    if (tile.type !== 'wan' && tile.type !== 'tiao' && tile.type !== 'tong') {

### 5.2 getChiOptions(tile, hand, fromPlayerIndex)

**功能**: 获取吃牌选项

**代码位置**: server.js:1530-1581

**返回值**: ChiOption[]

```javascript
getChiOptions(tile, hand, fromPlayerIndex = null) {
    const options = [];
    const value = tile.value;
    const type = tile.type;
    
    // 情况1：吃边张（value-2, value-1）← tile
    // 例如：手牌有1、2万，吃3万成123万
    if (value >= 3) {
        const left1 = hand.find(t => t.type === type && t.value === value - 2);
        const left2 = hand.find(t => t.type === type && t.value === value - 1);
        if (left1 && left2) {
            options.push({
                type: 'chi',
                tiles: [left1, left2, tile],
                pattern: '边张',
                from: fromPlayerIndex
            });
        }
    }
    
    // 情况2：吃嵌张（value-1, value, value+1）
    // 例如：手牌有1、3万，吃2万成123万
    if (value >= 2 && value <= 8) {
        const mid = hand.find(t => t.type === type && t.value === value - 1);
        const right = hand.find(t => t.type === type && t.value === value + 1);
        if (mid && right) {
            options.push({
                type: 'chi',
                tiles: [mid, tile, right],
                pattern: '嵌张',
                from: fromPlayerIndex
            });
        }
    }
    
    // 情况3：吃边张（value, value+1, value+2）→
    // 例如：手牌有8、9万，吃7万成789万
    if (value <= 7) {
        const right1 = hand.find(t => t.type === type && t.value === value + 1);
        const right2 = hand.find(t => t.type === type && t.value === value + 2);
        if (right1 && right2) {
            options.push({
                type: 'chi',
                tiles: [tile, right1, right2],
                pattern: '边张',
                from: fromPlayerIndex
            });
        }
    }
    
    return options;
}
```

## 六、暗杠处理

### 6.1 checkAnGangAvailable(player)

**功能**: 检查玩家是否有暗杠机会

**代码位置**: server.js:2150-2173

```javascript
checkAnGangAvailable(player) {
    const anGangActions = [];
    const tileCounts = {};
    const tileMap = {};
    
    player.hand.forEach(t => {
        const key = `${t.type}_${t.value}`;
        tileCounts[key] = (tileCounts[key] || 0) + 1;
        if (!tileMap[key]) tileMap[key] = [];
        tileMap[key].push(t);
    });
    
    for (const key in tileCounts) {
        if (tileCounts[key] === 4) {
            const tiles = tileMap[key];
            anGangActions.push({
                tile: tiles[3],
                tiles: tiles
            });
        }
    }
    
    return anGangActions;
}
```

### 6.2 executeAnGang(player, anGangOptions)

**功能**: 执行暗杠

**代码位置**: server.js:2043-2147

```javascript
executeAnGang(player, anGangOptions) {
    if (!anGangOptions || anGangOptions.length === 0) return;
    
    const option = anGangOptions[0];
    const sameTiles = option.tiles;
    
    // 从手牌中移除这4张牌
    sameTiles.forEach(t => {
        const idx = player.hand.findIndex(h => h.id === t.id);
        if (idx !== -1) player.hand.splice(idx, 1);
    });
    
    // 添加到副露
    player.melds.push({
        type: 'gang',
        tiles: sameTiles,
        from: player.seatIndex  // 暗杠的from是自己
    });
    
    this.broadcast('action_executed', {
        playerIndex: player.seatIndex,
        action: 'an_gang',
        tile: sameTiles[0],
        tileName: getTileName(sameTiles[0])
    });
    
    // 杠后摸一张牌
    this.gameState.currentPlayerIndex = player.seatIndex;
    this.gameState.turnPhase = 'draw';
    this.gameState.gangShangPao = true;
    
    this.broadcastGameState();
    
    // 延迟摸牌
    setTimeout(() => {
        if (this.gameRunning) {
            const newTile = this.drawTileForPlayer(player, false);
            if (!newTile) {
                this.endRound('draw', -1, -1, false, false);
                return;
            }
            
            this.gameState.lastDrawnTile = newTile;
            this.gameState.turnPhase = 'discard';
            
            // 检查是否自摸
            if (player.isQiao && this.canHu(player.hand, player.melds)) {
                // ... 提示自摸
            } else {
                // 检查是否还有其他暗杠机会
                const remainingAnGangActions = this.checkAnGangAvailable(player);
                if (remainingAnGangActions.length > 0) {
                    // ... 提示继续暗杠
                } else {
                    this.notifyCurrentPlayer();
                }
            }
        }
    }, 300);
}
```

## 九、三口计数

每次吃/碰/杠时，更新三口计数：

```javascript
// 更新三口计数
const discardPlayerIndex = this.gameState.lastDiscardPlayer;
player.sankouCounts[discardPlayerIndex]++;

// 三口计数用于包三口规则
// 如果玩家A对玩家B吃/碰/杠达到3次，则形成三口关系
// 自摸时：B胡牌，A单独赔付5份
// 点炮时：B胡A的牌，A赔付2份
```
