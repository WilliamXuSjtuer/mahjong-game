# 计分系统

本文档详细描述麻将计分系统的实现，包括番型计算、花数计算、苍蝇分计算等。

## 一、计分概述

### 1.1 计分公式

```
基础分 = 花数 × 2^番数
总分 = 基础分 + 苍蝇分
荒番翻倍 = 总分 × 2^荒番数
杠上炮 = 点炮分 × 3
封顶 = 50分
```

### 1.2 计分流程

```
┌─────────────────────────────────────────────────────────────────────┐
│                        计分流程                                      │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│  1. 计算番数 (calculateFan)                                         │
│     ├── 检测门清                                                    │
│     ├── 检测七对子                                                  │
│     ├── 检测碰碰胡                                                  │
│     ├── 检测清一色/混一色                                           │
│     ├── 检测杠开                                                    │
│     └── 检测大单吊                                                  │
│                                                                     │
│  2. 计算花数 (calculateHua)                                         │
│     ├── 底花（1花）                                                 │
│     ├── 花牌（每张1花）                                             │
│     ├── 杠/刻子花数                                                 │
│     └── 风向额外花数                                                │
│                                                                     │
│  3. 计算苍蝇分 (calculateCangying)                                  │
│     ├── 检查是否有资格                                              │
│     └── 根据苍蝇牌计算分值                                          │
│                                                                     │
│  4. 计算最终分数 (calculateScore)                                   │
│     ├── 应用计分公式                                                │
│     ├── 荒番翻倍                                                    │
│     ├── 杠上炮翻倍                                                  │
│     ├── 封顶处理                                                    │
│     └── 计算各玩家分数变化                                          │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

## 二、番数计算

### 2.1 calculateFan(player, isZimo, isGangKai)

**功能**: 计算番型和番数

**代码位置**: server.js:2763-2832

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| player | Player | 胡牌玩家 |
| isZimo | boolean | 是否自摸 |
| isGangKai | boolean | 是否杠开 |

**返回值**: `{fanList: FanItem[], totalFan: number}`

```javascript
calculateFan(player, isZimo = false, isGangKai = false) {
    const hand = player.hand;
    const melds = player.melds;
    
    let fanList = [];
    let totalFan = 0;
    const isMenQing = melds.length === 0;
    
    // 1. 门清（1番）
    if (isMenQing) {
        fanList.push({ name: '门清', fan: 1 });
        totalFan += 1;
    }
    
    // 2. 自摸（0番）
    if (isZimo) {
        fanList.push({ name: '自摸', fan: 0 });
        totalFan += 0;
    }
    
    // 3. 七对子（2番）
    if (this.isQiDui(hand)) {
        fanList.push({ name: '七对子', fan: 2 });
        totalFan += 2;
    }
    
    // 4. 碰碰胡（2番）
    const isPengPengHuFlag = this.checkPengPengHu(hand, melds);
    if (isPengPengHuFlag) {
        fanList.push({ name: '碰碰胡', fan: 2 });
        totalFan += 2;
    }
    
    // 5. 清一色（3番）
    if (this.isQingYiSe(hand, melds)) {
        fanList.push({ name: '清一色', fan: 3 });
        totalFan += 3;
    }
    // 6. 混一色（2番）
    else if (this.isHunYiSe(hand, melds)) {
        fanList.push({ name: '混一色', fan: 2 });
        totalFan += 2;
    }
    
    // 7. 杠开（1番）
    if (isGangKai) {
        fanList.push({ name: '杠开', fan: 1 });
        totalFan += 1;
    }
    
    // 8. 大单吊（1番）
    if (this.isDaDanDiao(hand, melds)) {
        fanList.push({ name: '大单吊', fan: 1 });
        totalFan += 1;
    }
    
    // 基本胡（0番）
    if (totalFan === 0) {
        fanList.push({ name: '基本胡', fan: 0 });
    }
    
    return { fanList, totalFan };
}
```

### 2.2 番型一览表

| 番型 | 番数 | 条件 | 说明 |
|------|------|------|------|
| 门清 | 1 | 无吃碰杠 | 没有副露 |
| 自摸 | 0 | 自己摸牌胡 | 基础属性 |
| 七对子 | 2 | 7个对子 | 特殊牌型 |
| 碰碰胡 | 2 | 4刻子+1对 | 全刻子 |
| 清一色 | 3 | 单一花色 | 无字牌 |
| 混一色 | 2 | 单一花色+字牌 | 有字牌 |
| 杠开 | 1 | 杠后摸牌胡 | 杠上开花 |
| 大单吊 | 1 | 手牌2张 | 单钓将 |
| 基本胡 | 0 | 无特殊牌型 | 基础胡牌 |

## 三、花数计算

### 3.1 calculateHua(player)

**功能**: 计算花数

**代码位置**: server.js:2895-2958

**返回值**: `{huaList: HuaItem[], totalHua: number}`

```javascript
calculateHua(player) {
    let huaList = [];
    let totalHua = 1;  // 底花1花
    huaList.push({ name: '底花', hua: 1 });
    
    // 花牌（每张1花）
    const flowerCount = player.flowers ? player.flowers.length : 0;
    if (flowerCount > 0) {
        huaList.push({ name: `花牌×${flowerCount}`, hua: flowerCount });
        totalHua += flowerCount;
    }
    
    const isWindTile = (tile) => tile && tile.type === 'wind';
    
    // 统计杠和刻子
    for (const meld of player.melds) {
        if (meld.type === 'gang') {
            const isWind = meld.tiles && meld.tiles.length > 0 && 
                          isWindTile(meld.tiles[0]);
            
            if (meld.from !== undefined && meld.from !== player.seatIndex) {
                // 明杠
                if (isWind) {
                    huaList.push({ name: '风向明杠', hua: 2 });
                    totalHua += 2;
                } else {
                    huaList.push({ name: '明杠', hua: 1 });
                    totalHua += 1;
                }
            } else {
                // 暗杠
                if (isWind) {
                    huaList.push({ name: '风向暗杠', hua: 3 });
                    totalHua += 3;
                } else {
                    huaList.push({ name: '暗杠', hua: 2 });
                    totalHua += 2;
                }
            }
        } else if (meld.type === 'peng') {
            const isWind = meld.tiles && meld.tiles.length > 0 && 
                          isWindTile(meld.tiles[0]);
            if (isWind) {
                huaList.push({ name: '风向刻子', hua: 1 });
                totalHua += 1;
            }
        }
    }
    
    // 手牌中的暗刻（风向）
    if (player.hand) {
        const tileCounts = {};
        for (const tile of player.hand) {
            const key = `${tile.type}_${tile.value}`;
            tileCounts[key] = (tileCounts[key] || 0) + 1;
        }
        for (const tile of player.hand) {
            const key = `${tile.type}_${tile.value}`;
            if (tileCounts[key] >= 3 && isWindTile(tile)) {
                huaList.push({ name: '风向暗刻', hua: 2 });
                totalHua += 2;
                delete tileCounts[key];
                break;
            }
        }
    }
    
    return { huaList, totalHua };
}
```

| 类型 | 花数 | 说明 |
|------|------|------|
| 底花 | 1 | 胡牌基础花 |
| 花牌 | 每张1花 | 春夏秋冬梅兰竹菊 |
| 明杠 | 1 | 别人打的牌杠 |
| 暗杠 | 2 | 自己手牌杠 |
| 风向明杠 | 2 | 东南西北的明杠 |
| 风向暗杠 | 3 | 东南西北的暗杠 |

## 四、苍蝇分计算

### 4.1 calculateCangying(player, cangyingTile, fanResult)

**功能**: 计算苍蝇分

**代码位置**: server.js:2961-3024

**说明**: 胡牌后从牌墙翻一张作为苍蝇牌，有特定牌型可获得额外分数

```javascript
calculateCangying(player, cangyingTile, fanResult = null) {
    const cangyingList = [];
    let totalCangying = 0;
    
    if (!cangyingTile) {
        return { cangyingList, totalCangying };
    }
    
    if (!fanResult) {
        fanResult = this.calculateFan(player, false, false);
    }
    
    // 检查是否有资格飞苍蝇
    const hasQualifyingFan = fanResult.fanList.some(f => 
        ['门清', '混一色', '清一色', '七对子', '大单吊', '碰碰胡'].includes(f.name)
    );
    
    if (!hasQualifyingFan) {
        return { cangyingList, totalCangying };
    }
    
    // 计算苍蝇分值
    let cangyingValue = 0;
    let cangyingName = '';
    
    // 数字牌
    if (cangyingTile.type === 'wan' || 
        cangyingTile.type === 'tiao' || 
        cangyingTile.type === 'tong') {
        
        const typeName = cangyingTile.type === 'wan' ? '万' : 
                        cangyingTile.type === 'tong' ? '筒' : '条';
        
        if (cangyingTile.value === 1) {
            cangyingValue = 10;  // 一万/一条/一筒 = 10分
        } else {
            cangyingValue = cangyingTile.value;  // 其他 = 牌面值
        }
        cangyingName = `苍蝇：${cangyingTile.value}${typeName}`;
    } 
    // 风牌
    else if (cangyingTile.type === 'wind') {
        cangyingValue = 5;
        cangyingName = `苍蝇：${WIND_NAMES[cangyingTile.value] || '风牌'}`;
    } 
    // 箭牌
    else if (['zhong', 'fa', 'bai'].includes(cangyingTile.type)) {
        cangyingValue = 5;
        const tileNames = { zhong: '中', fa: '发', bai: '白' };
        cangyingName = `苍蝇：${tileNames[cangyingTile.type]}`;
    } 
    // 花牌
    else if (cangyingTile.type === 'flower') {
        cangyingValue = 5;
        cangyingName = `苍蝇：${FLOWER_NAMES[cangyingTile.value] || '花牌'}`;
    }
    
    if (cangyingValue > 0) {
        cangyingList.push({ name: cangyingName, value: cangyingValue });
        totalCangying = cangyingValue;
    }
    
    return { cangyingList, totalCangying };
}
```

### 4.2 苍蝇分值表

| 苍蝇牌 | 分值 | 说明 |
|--------|------|------|
| 一万/一条/一筒 | 10 | 幺九牌 |
| 二万~九万 | 2-9 | 牌面值 |
| 二条~九条 | 2-9 | 牌面值 |
| 二筒~九筒 | 2-9 | 牌面值 |
| 风牌（东南西北） | 5 | 固定值 |
| 箭牌（中发白） | 5 | 固定值 |
| 花牌 | 5 | 固定值 |

### 4.3 苍蝇资格

只有以下牌型才能飞苍蝇：
- 门清
- 混一色
- 清一色
- 七对子
- 大单吊
- 碰碰胡

## 五、最终分数计算

### 5.1 calculateScore(...)

**功能**: 计算最终分数和各玩家分数变化

**代码位置**: server.js:3027-3132

**参数**:
| 参数 | 类型 | 说明 |
|------|------|------|
| winner | Player | 胡牌玩家 |
| loserIndex | number | 点炮者索引 |
| fanResult | object | 番型结果 |
| huaResult | object | 花数结果 |
| cangyingResult | object | 苍蝇结果 |
| isZimo | boolean | 是否自摸 |
| isHuangFanRound | boolean | 是否荒番局 |
| huangFanCount | number | 荒番数 |
| isGangShangPao | boolean | 是否杠上炮 |

**返回值**: ScoreResult

```javascript
calculateScore(winner, loserIndex, fanResult, huaResult, cangyingResult, 
               isZimo, isHuangFanRound = false, huangFanCount = 0, 
               isGangShangPao = false) {
    
    const MAX_SCORE = 50;  // 封顶50分
    
    // 分数 = 花数 × 2^番数 + 苍蝇分
    const baseScore = huaResult.totalHua * Math.pow(2, fanResult.totalFan);
    const totalBeforeHuangFan = baseScore + cangyingResult.totalCangying;
    
    // 荒番翻倍
    let finalScore = totalBeforeHuangFan;
    if (isHuangFanRound && huangFanCount > 0) {
        finalScore = totalBeforeHuangFan * Math.pow(2, huangFanCount);
    }
    
    // 杠上炮：点炮分数 × 3
    if (isGangShangPao) {
        finalScore = finalScore * 3;
    }
    
    // 封顶
    finalScore = Math.min(finalScore, MAX_SCORE);
    
    const scoreChanges = [0, 0, 0, 0];
    let sankouInfo = null;
    
    // 检查三口关系
    const checkSankou = (playerIndex, targetIndex) => {
        return this.players[playerIndex].sankouCounts[targetIndex] >= 3;
    };
    
    if (isZimo) {
        // 自摸：检查是否有包三口关系
        let sankouPlayer = -1;
        for (let i = 0; i < 4; i++) {
            if (i !== winner.seatIndex && checkSankou(winner.seatIndex, i)) {
                sankouPlayer = i;
                sankouInfo = {
                    type: 'zimo',
                    sankouPlayer: i,
                    sankouCount: this.players[winner.seatIndex].sankouCounts[i],
                    message: `${this.players[i].username} 与 ${winner.username} 有三口关系，单独赔付 5 份`
                };
                break;
            }
        }
        
        if (sankouPlayer !== -1) {
            // 包三口：该玩家单独付5份
            scoreChanges[winner.seatIndex] = finalScore * 5;
            scoreChanges[sankouPlayer] = -finalScore * 5;
        } else {
            // 正常自摸：三家各付分数
            for (let i = 0; i < 4; i++) {
                if (i === winner.seatIndex) {
                    scoreChanges[i] = finalScore * 3;  // 赢家得3份
                } else {
                    scoreChanges[i] = -finalScore;     // 其他各付1份
                }
            }
        }
    } else {
        // 点炮：检查胡牌者与放炮者是否有三口关系
        const hasSankou = checkSankou(winner.seatIndex, loserIndex);
        
        if (hasSankou) {
            // 包三口：放炮者付2份
            scoreChanges[winner.seatIndex] = finalScore * 2;
            scoreChanges[loserIndex] = -finalScore * 2;
            sankouInfo = {
                type: 'dianpao',
                sankouPlayer: loserIndex,
                sankouCount: this.players[winner.seatIndex].sankouCounts[loserIndex],
                message: `${this.players[loserIndex].username} 与 ${winner.username} 有三口关系，赔付 2 份`
            };
        } else {
            // 正常点炮：放炮者付1倍
            scoreChanges[winner.seatIndex] = finalScore;
            scoreChanges[loserIndex] = -finalScore;
        }
    }
    
    return {
        baseScore,
        cangyingScore: cangyingResult.totalCangying,
        finalScore,
        scoreChanges,
        fanDetail: fanResult.fanList,
        huaDetail: huaResult.huaList,
        cangyingDetail: cangyingResult.cangyingList,
        totalFan: fanResult.totalFan,
        totalHua: huaResult.totalHua,
        isHuangFanRound,
        huangFanCount,
        huangFanMultiplier: isHuangFanRound && huangFanCount > 0 ? 
                           Math.pow(2, huangFanCount) : 1,
        isGangShangPao,
        sankouInfo
    };
}
```

## 六、计分示例

### 示例1: 基本胡

```
手牌: [一万, 一万, 二万, 三万, 四万, 五万, 六万, 七万, 八万, 九万, 东风, 东风, 东风, 东风]
副露: 无
花牌: 春

番型: 门清(1番)
花数: 底花(1) + 花牌(1) = 2花
苍蝇: 无资格

计算: 2 × 2^1 = 4分
```

### 示例2: 清一色碰碰胡

```
手牌: [一万, 一万, 三万, 三万, 五万, 五万, 七万, 七万, 九万, 九万]
副露: [{碰: 一万}, {碰: 三万}]
花牌: 春、夏

番型: 碰碰胡(2番)
花数: 底花(1) + 花牌(2) = 3花
苍蝇: 一万(10分)

计算: 3 × 2^2 + 10 = 22分
```

### 示例3: 荒番局

```
番型: 清一色(3番)
花数: 5花
苍蝇: 8分
荒番数: 2

计算: (5 × 2^3 + 8) × 2^2 = 48 × 4 = 192 → 封顶50分
```

## 七、包三口规则

### 7.1 三口形成条件

同一玩家对另一玩家吃/碰/杠达到3次及以上。

### 7.2 三口效果

| 情况 | 效果 |
|------|------|
| 自摸 | 包三口者单独赔付5份，其他玩家无关 |
| 点炮 | 包三口者赔付2份（正常点炮是1份） |

### 7.3 三口计数

```javascript
// 每次吃/碰/杠时更新
player.sankouCounts[discardPlayerIndex]++;

// 检查三口
const hasSankou = player.sankouCounts[targetIndex] >= 3;
```

## 八、特殊规则

### 8.1 荒番

- 流局后荒番数+1
- 荒番局分数 × 2^n（n为荒番数）
- 有人胡牌后荒番数清零

### 8.2 杠上炮

- 杠牌后补牌再打出点炮
- 点炮分数 × 3

### 8.3 封顶

- 单局最高50分
